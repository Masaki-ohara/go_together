# サービス概要
遊びの計画を立てるのが苦手な人に向けて
遊びの計画を立てやすくするサービス

# メインのターゲットユーザー
遊びの計画を立てる人が苦手な人向け
次どこ行こうか迷う人向け

# ユーザーが抱える課題
遊びの計画を立てるのが意外と時間がかかる
無計画で遊びに行くと次どこに行くか迷いやすい

### 【サービス概要】
遊びの計画をタイムテーブル通りに遊ぶためのアプリ


### ログイン前にできること

---
個人プラン(ソロ活用)を立てられる

### ログイン後にできること

---

グループを作ってグループの計画を立てられる
プランをいくつか挙げて投票できるようになります。
